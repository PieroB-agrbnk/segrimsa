# SEGRIMSA - Registro de Catequesis

Formulario digital para registro de catequesis (Primera Comunion, Confirmacion, Bautizo).

Reemplaza las hojas de papel. Los padres escanean un QR y llenan el formulario desde su celular.

## Deploy

Este proyecto esta configurado para Streamlit Community Cloud.

- Main file: `formulario.py`
- Python version: 3.12
